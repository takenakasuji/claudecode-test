# サブスクリプション分析ダッシュボード

モバイルアプリのサブスクリプション契約状況を可視化するダッシュボードです。

## 機能

- **主要メトリクスの表示**
  - 有効サブスクリプション数
  - 新規契約数
  - 解約数
  - MRR（月次経常収益）
  - トライアル転換率

- **柔軟なフィルタリング**
  - 期間（過去7日間 / 30日間 / 90日間 / 1年間）
  - プラットフォーム（iOS / Android / 全体）
  - プランタイプ（月額 / 年額 / 全体）

- **インタラクティブなグラフ**
  - 有効サブスクリプション数の推移
  - 新規契約 vs 解約の比較
  - MRRの推移
  - トライアル転換数の推移

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# プレビュー
npm run preview
```

## 技術スタック

- **フレームワーク**: React 18 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS
- **グラフ**: Recharts
- **日付処理**: date-fns

## データについて

このダッシュボードは、リアルなトレンドとスパイクを含むダミーデータを使用しています。

- 成長トレンド（時間経過とともに増加）
- 週次パターン（週末の変動）
- 季節性のスパイク（年末年始、ブラックフライデー、夏季プロモーションなど）
- ランダムなミニスパイク（キャンペーンイベントをシミュレート）
