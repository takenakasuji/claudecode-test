# サブスクリプション分析ダッシュボード

モバイルアプリのサブスクリプション契約状況をリアルタイムで可視化するモダンなダッシュボードアプリケーションです。

## 機能

### KPI指標
- 📊 **有効サブスク数**: 現在アクティブなサブスクリプションの総数
- 🎉 **新規サブスク**: 期間内の新規契約数
- 😞 **解約数**: 期間内の解約数
- 💰 **MRR**: 月次経常収益（Monthly Recurring Revenue）
- 🎯 **トライアル転換率**: 無料トライアルから有料プランへの転換率

### フィルタリング機能
- **期間指定**: 開始日と終了日を選択して分析期間を設定
- **プラットフォーム**: iOS、Android、または全体で絞り込み
- **プランタイプ**: 月額プラン、年額プラン、または全体で絞り込み

### インタラクティブグラフ
- **サブスクリプション推移**: 有効サブスク、新規、解約の時系列チャート
- **MRR推移**: 月次経常収益の推移を棒グラフで表示
- **プラットフォーム別分布**: iOS vs Androidの比較（円グラフ）
- **プラン別分布**: 月額 vs 年額の比較（円グラフ）
- **インサイト**: 自動生成される分析コメント

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - 型安全性
- **Vite** - 高速ビルドツール
- **Tailwind CSS** - モダンなスタイリング
- **Recharts** - インタラクティブなチャート
- **date-fns** - 日付処理

## セットアップ

### 依存関係のインストール
```bash
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```

ブラウザで http://localhost:5173 を開いてダッシュボードを表示します。

### プロダクションビルド
```bash
npm run build
```

ビルドされたファイルは `dist/` ディレクトリに生成されます。

### プレビュー
```bash
npm run preview
```

## データについて

このダッシュボードは**ダミーデータ**を使用しています。データには以下の特徴があります：

- リアルな成長トレンド（徐々に増加）
- 週末効果（土日は新規契約が減少）
- ランダムなスパイク（キャンペーン等を想定）
- ノイズ（現実的な変動）

実際のプロダクション環境では、`src/dataGenerator.ts` を実際のAPIやデータベースからのデータ取得に置き換えてください。

## プロジェクト構造

```
src/
├── components/          # Reactコンポーネント
│   ├── Charts.tsx      # グラフコンポーネント
│   ├── FilterPanel.tsx # フィルターUI
│   └── MetricCard.tsx  # KPIカード
├── dataGenerator.ts    # ダミーデータ生成
├── dataUtils.ts        # データ処理・集計
├── types.ts           # TypeScript型定義
├── App.tsx            # メインアプリ
├── main.tsx           # エントリーポイント
└── index.css          # グローバルスタイル
```

## カスタマイズ

### データソースの変更
`src/dataGenerator.ts` を編集して、実際のAPIエンドポイントからデータを取得するように変更できます。

### スタイルのカスタマイズ
`tailwind.config.js` でカラーテーマやその他のデザイントークンをカスタマイズできます。

### 新しいメトリクスの追加
1. `src/types.ts` に新しい型を追加
2. `src/dataUtils.ts` に集計ロジックを追加
3. `src/components/` に新しいコンポーネントを作成
4. `src/App.tsx` でコンポーネントを統合
